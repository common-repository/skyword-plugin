Allows integration with the skyword publishing platform.

== Description ==

[Skyword360](http://www.skyword.com/)  is the leading comprehensive content production platform. Skyword360 enables brands, retailers, media companies, and agencies to acquire and engage customers by efficiently producing quality content optimized for search and the social web. Quality content is essential for reaching and engaging consumers today, but the creation process is messy, inconsistent and immeasurable. Skyword360 makes it easy to produce, optimize, and promote content at any scale to create meaningful, lasting relationships with customers.
